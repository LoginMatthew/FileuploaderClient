About the File Uploader program

People are able to store files and check the uploaded files and able to download them which are uploaded. The layout of the pages is also optimized for mobile use (Tested on Mozilla Firefox 121.0.1 (desktop) and google chrome (mobile)). This program is perfect to use to share files between family members at home.

Usage of the site

Uploaded files can be watch on the site if those files are pictures or videos or audio files (minimum file size > 0byte). There are three type of users which are guest, user and admin. The uplodaed files can be set to be public or private (only the uploader and admintrators are able to see). Only the file's owner and the admin are able to delete and modify the visibilty of the selected content. Any User is able to upgrade a Guest to "user" level but unable to modify User to "guest" level.

On "All Uploaded data" page, If you tap/click on picture/video/audio then a modal (popup) will be opened to see the content. To exit from the modal then click/tap outside the modal (popup) and the modal will be closed.

Development information

The "data objects" are encrypted and transfered between the server and client. Files are not ecrypted. Password are stored with ecryption. Token is used to avoid unauthrize access so only logged in user is able to requerst data from the server. The refresh token is used not to give a long period of Token. Automatic logout is done after a given period of time or inactivity.

The client program mainly tested on Desktop Mozilla Firefox and some part was tested on Mobile's Google Chrome!

The application development is NOT finished and there are some known errors or the way what I longed or it was requested to develop in a given case!
